Source code

  • Magisk Delta is open source under GNU v3. The source-code.zip will be published along with released build here: https://github.com/HuskyDG/magisk-files/releases
  • The process to build Magisk Delta is same as Official Magisk. Remember to add -delta prefix into versionName in config.prop.
